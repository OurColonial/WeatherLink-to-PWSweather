PWSweather to Davis WeatherLink API Key Mapping

# PWSWeather Davis WeatherLink Required Comment
1 ID Station ID As Registered YES Station ID as registered
2 PASSWORD API KEY YES Station specific API Key from Admin page
3 dateutc Current UTC Date YES Date and time in the format of year-mo-da+hour:min:sec
4 winddir wind_dir_last NO Wind direction in degrees
5 windspeedmph wind_speed_last NO Wind speed in miles per hour
6 windgustmph wind_dir_at_hi_speed_last_10_min NO Wind gust in miles per hour
7 tempf temp NO Temperature in degrees fahrenheit
8 rainin rainfall_last_60_min NO Hourly rain in inches
9 dailyrainin rainfall_last_24_hr NO Daily rain in inches
10 monthrainin NO Monthly rain in inches
11 yearrainin NO Seasonal rain in inches (usually local meteorological year)
12 baromin bar_sea_level NO Barometric pressure in inches
13 dewptf dew_point NO Dew point in degrees fahrenheit
14 humidity hum NO Humidity in percent
15 weather NULL NO Current weather or sky conditions using standard METAR abbreviations and intensity (e.g. -RA, +SN, SKC, etc.)
16 solarradiation solar_rad NO Solar radiation
17 UV NULL NO UV
18 softwaretype DavisWeatherLinkLive YES Software type
19 action updateraw YES always concludes with action=updateraw to indicate the end of the readings
